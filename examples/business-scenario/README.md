# Business scenario: sales performance

Run the automated scenario:

```bash
# From repo root: ensure server is running (make start-docker or make start-local)
cd examples/business-scenario
bash run-business-scenario.sh
```

Prerequisites: PgQueryNarrative server running, database with demo data. See [Quick start](../../docs/getting-started/quickstart.md) and [API examples](../../docs/api/examples.md).
